product_lm_comparator — поиск и сравнение поставщиков продуктов питания с LLM-ассистентом.

Архитектура: два процесса — FastAPI бекенд (REST API, SQLAlchemy, PostgreSQL) и Streamlit фронтенд. Фронтенд обращается к бекенду по HTTP. Локальная LLM (Ollama) используется для чата-сравнения поставщиков и для парсинга сайтов поставщиков (извлечение структурированных данных через LLM).

Основные возможности:
- Каталог поставщиков с фильтрацией по категориям, городу, рейтингу
- Управление контактами, сертификатами, условиями заказа, заметками
- Диалоговое сравнение поставщиков с LLM-ассистентом
- Парсинг сайта поставщика с авто-извлечением данных
- REST API со Swagger-документацией (/docs)
- Docker Compose (PostgreSQL + бекенд + фронтенд)
